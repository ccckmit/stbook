指數分布 (Exponential Distribution)

  • 範圍：x=0,…,k （取出 x 個白球）
  • 意義：白球有 m 個，黑球 n 個，取出 k 個球，其中有 x 個白球的機率; (取後不放回)
  • R 函數： [[ ]]
  • http://stat.ethz.ch/R-manual/R-patched/library/stats/html/Hypergeometric.html

特性

動差生成函數：

R 程式範例一

op=par(mfrow=c(2,2))
curve(dexp(x, 1), 0, 10)
curve(dexp(x, 5), 0, 10)
curve(dexp(x, 1/5), 0, 10)
curve(dexp(x, 1/2), 0, 10)

R 程式範例二

> dexp(1) - exp(-1) #-> 0
[1] 0
